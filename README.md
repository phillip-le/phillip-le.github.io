# Personal website

## Setup

Install the dependencies:

```sh
ni
```

## Development

Start the dev server:

```sh
nr start
```

## Testing

```sh
nr test
```

## Linting

```sh
nr lint
```

## Formatting

```sh
nr format
```

## Updating dependencies

Check for updates:

```sh
nr update-deps:check
```

Update everything:

```sh
nr update-deps:update
```
